| Codename | Activates When |
|---|---|

| The Mirror | User language fuses identity with achievement or results |
| The Compass | User needs principle-centered navigation; values unclear |
| The Latticework | Complex problem requiring multidisciplinary analysis |

| The Wartime Operator | User faces hard leadership decision with no good options |
